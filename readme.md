dotfiles
========

These are my dot files.

Put a symbolic link into `conf.avail` directory to enable a config

```
$ ./install.py
```

If no file is found, a symbolic link is created.

## xinitrc

Mandatory packages

* app-admin/conky
* x11-apps/xsetroot
* x11-wm/dwm
* x11-terms/xterm

install `conky` without `X` support
